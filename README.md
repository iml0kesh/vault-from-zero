# Vault From Zero

starting from basics to real-world DevOps and security use cases.

- Learn Vault from scratch

- Practice real CLI-based workflows
- Build Vault + Jenkins / Kubernetes use cases

## Learning Style

- Beginner-friendly notes

## Tech Stack

- HashiCorp Vault

- CLI-based learning
